# Expense Manager

A robust, full-stack web application designed to help users track, analyze, and manage their personal finances effectively. 

## 🚀 Overview

Expense Manager provides a seamless experience for logging daily transactions, categorizing expenses, and maintaining a clear overview of financial health. Built on the MERN stack, the application features a responsive, intuitive interface tailored for optimal user experience and detailed financial tracking.

## ✨ Features

- **User Authentication:** Secure signup and login functionality.
- **Transaction Management:** Easily add, edit, and delete income and expense records.
- **Categorization:** Group transactions by categories (e.g., Food, Travel, Utilities, Salary).
- **Interactive Dashboard:** Visualize financial data with summaries and graphical representations.
- **Responsive Design:** Fully optimized for both desktop and mobile viewing.

## 🛠️ Tech Stack

**Frontend:**
- **React.js**: Core UI library for building the dynamic single-page application.
- **Bootstrap & Pure CSS**: Grid layouts and responsive styling. Custom CSS is utilized extensively for fine-grained design control to maintain a clean architecture without Tailwind.

**Backend:**
- **Node.js & Express.js**: RESTful API architecture handling business logic, routing, and client requests.
- **MongoDB**: NoSQL database for flexible and scalable data storage.
- **Mongoose**: Object Data Modeling (ODM) for robust project schemas and validation.

**Design & Prototyping:**
- **Canva**: Utilized for UI screens, layout planning, and visual asset generation prior to development.
